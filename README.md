# VERBTW FILMS

Личная коллекция просмотренных фильмов и сериалов VERBTW.

## Возможности

- 312 фильмов и сериалов с поиском, жанрами и режиссёрами
- отдельная страница каждого фильма
- постеры и описания из открытых источников
- встроенный предпросмотр трейлера при наведении
- личные оценки, сохраняемые в браузере

## Локальный запуск

```bash
python3 server.py
```

После запуска сайт доступен по адресу `http://localhost:8000`.
